02 · Ключевые возможности

Что получит бизнес и клиент

Платформа решает четыре задачи одновременно: витрина для клиента, рабочий инструмент для менеджеров, канал лидогенерации и база знаний для ИИ.
Каталог

Единый каталог с маркерной классификацией L1 → L2 → L3

Одна номенклатурная единица относится сразу к нескольким разделам: тип подвижного состава → классификация депо → цех/участок. Фасетные фильтры без перезагрузки страницы. Источник истины — 1С.

QR + 1С

QR-код на каждое изделие

Клиент сканирует код прямо на станке, попадает на карточку с актуальным описанием, ценой и кнопкой заявки. Формируется на стороне 1С.

«Мой каталог»

Подборка без регистрации: клиент собирает нужное → получает автоматическое базовое ТКП на почту, менеджер — уведомление и лид в 1С.

Консультация в один клик

Форма обращения с привязкой к странице каталога + быстрый переход в Telegram, WhatsApp, МАКС. Защита Яндекс.Капчей.

Цифровое депо

Раздел, презентующий готовые комплексы оборудования для разных типов депо — от предварительного подбора к заявке на проектирование.

Интеграция

Двусторонний обмен с 1С через CommerceML

Номенклатура, базовые цены, QR-коды, признаки выгрузки, статус «временно не производится», фотографии — всё синхронизируется автоматически по расписанию. Заявки и лиды уходят в 1С обратно.

hh.ru · Avito · Telegram

Вакансии тянутся с hh.ru автоматически. Товары публикуются на Avito из 1С. Новости — в Telegram и МАКС одной кнопкой из админки.

Мониторинг и журнал ошибок

Дашборд метрик, алерты по сбоям интеграций, соответствие 152-ФЗ. Система ролей, журнал действий, личный кабинет клиента.

04 · ИИ-модуль платформы

ИИ, обученный на вашей документации

Корпоративная база знаний + RAG-архитектура дают ответам опору на реальные документы НПЦ ПРОМТЕХ. Никаких галлюцинаций — только выдержки из паспортов, сертификатов и ТУ.
ИИ
Консультант НПЦ ПРОМТЕХ
● на связи
Добрый день. Ищу колёсотокарный станок для трамвайного депо. Что можете предложить?
Для трамвайного депо у нас есть два подходящих решения. Оптимальный вариант — станок КЖ-30Т (децимальный № НПТ.412.17). Диаметр обработки до 1250 мм, автоматическая диагностика профиля. Источник: Паспорт КЖ-30Т, п. 3.2 · Каталог НПЦ ПРОМТЕХ
Есть ли в нём функция неразрушающего контроля?
Да, в базовой комплектации. Модуль УЗК встроен в стойку ЧПУ, глубина контроля до 80 мм. Требует выделенной воздушной магистрали. Источник: ТУ 3185-014-НПТ-2024, раздел 4
Соберите мне подборку для полного участка обточки колёсных пар
Готово. Добавил 6 позиций в «Мой каталог»: станок, стенд дефектоскопии, подъёмники, подкатная тележка. Базовое ТКП отправлю на почту после заполнения контактов.

RAG-система на базе знаний

Корпоративные документы индексируются в векторное хранилище. Ответы генерируются на базе найденных фрагментов с guardrails — никакой отсебятины. LangChain.js / LlamaIndex.js.
